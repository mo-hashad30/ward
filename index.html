Install the Tool

💻 For PC / Mac Users

Make sure your Bookmarks Bar is visible (Ctrl+Shift+B on Windows, Cmd+Shift+B on Mac). Then, click and drag the blue button below directly onto your Bookmarks Bar.

Drag Me to Bookmarks

To use it, just click the bookmark whenever you are on the target website!

📱 For Mobile Users (Phones & Tablets)

Mobile browsers don't support drag-and-drop. Follow these 3 steps instead:

  1. Bookmark this page (or any webpage).
  2. Edit the bookmark: Change the Name to "Run Tool". Then, delete the website URL and paste the code from the box below into the URL field.
  3. To run it: Go to the website where the tool is meant to work. Type "Run Tool" in your browser's search/URL bar, and tap the bookmark suggestion that pops up.
